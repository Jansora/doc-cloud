# 14. 集群扩容

> 须确保安装了共同版本的 containerd , kubuadm 等

你可以在初始化的控制平面节点上运行以下命令来生成一个加入命令：

## 生成加入命令

```
kubeadm token create --print-join-command
```

## worker 节点加入
```
 kubeadm join 192.168.88.101:6443 --token ey3n4z.6rar3hm9fwbezcs4 --discovery-token-ca-cert-hash sha256:01df34acd184e43e5ed37fc292e30394588a7476a4b907149adfece88b2361d8
```

## control-plane 节点加入

使用上一步产生的命令，但是添加 --control-plane 标志来表示这是一个控制平面节点。

```
 kubeadm join 192.168.88.101:6443 --control-plane  --token ey3n4z.6rar3hm9fwbezcs4 --discovery-token-ca-cert-hash sha256:01df34acd184e43e5ed37fc292e30394588a7476a4b907149adfece88b2361d8
```

如果要加入 control-plane 节点, 需要确保 k8s 集群有稳定的 controlPlaneEndpoint 地址。
