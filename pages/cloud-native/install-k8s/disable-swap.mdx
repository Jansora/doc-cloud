## 禁用 swap 分区
编辑 `/etc/fstab`, 在 `/swap.img` 一行前面加 `#` 或者删除此行
```
# /swap.img     none    swap    sw      0       0
```

swappiness参数调整，修改`/etc/sysctl.d/99-kubernetes-cri.conf`添加下面一行：

swappiness参数调整，修改/etc/sysctl.d/99-kubernetes-cri.conf添加下面一行：

```
vm.swappiness=0
```
执行`sysctl -p /etc/sysctl.d/99-kubernetes-cri.conf`使修改生效。



重启
