# 免翻墙免费搭建 ChatGPT 中转服务

## 安装前说明
本项目搭建基于 https://github.com/pengzhile/pandora 的 serverless 版本 (https://github.com/pengzhile/pandora-cloud-serverless) 搭建.

部署于免费的托管平台 Vercel (https://vercel.com/)

可实现以下特点:
1. 免翻墙
2. 通过自己账号的 `access token` 访问, 与官网体验基本一致 (对话记录均可保存)
3. 无服务器架构 (托管于 Vercel ) , `access token` 存储于自己的浏览器本地, 安全
4. `access token` 只有 14 天. 到期后需要重新获取, 如何获取见 FAQ

## FAQ

### 不搭建, 该怎么免翻墙访问?

https://github.com/pengzhile/pandora 项目作者已提供一个  https://chat.zhile.io  共享地址, 可免费访问, 保险期间, 请自己搭建

### 如何获取  access token ?

> 见 https://github.com/pengzhile/pandora/blob/master/doc/wiki.md#%E5%85%B3%E4%BA%8E-access-token


### 更多文档请关注

1. https://github.com/pengzhile/pandora
2. https://github.com/pengzhile/pandora-cloud-serverless

### 补充说明

更加保险的做法是.
1. 在美区购买一个服务器, 一年只需几十 RMB, 部署代理 ChatGPT 服务.
2. 挂载ChatGPT 服务 到 Cloudflare tunnel 边缘节点, 本地挂载 Cloudflare tunnel 节点, 实现网络打通, 网络走 Cloudflare 中转, 非常稳定
3. 实现本地端口本地访问 ChatGPT
